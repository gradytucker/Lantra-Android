# Lantra - Android

Lantra is an Android application that allows you to discover devices as speakers on your local
network and stream audio from your device to them in real-time. 

## Features

#### Speaker Discovery

- Automatically detects devices on the same Wi-Fi network using mDNS/Bonjour.

#### Audio Streaming

- Supports stereo, 44.1kHz PCM streaming over TCP.

#### Player Controls

- control playback of currently playing audio
